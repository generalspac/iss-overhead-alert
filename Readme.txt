1)Environment Setup

Create a file named .env in the root of the project and add the following variables:
LAT=YOUR LATTITUDE
LONG=YOUR LONGITUDE
USER=your_email@gmail.com
PASSWORD=your_app_password
RECIPIENT_ADDR=recipient@gmail.com
RECIPIENT_NAME=RecipientName

2)Variable explanation
LAT → your latitude
LONG → your longitude
USER → the email address used to send the alert
PASSWORD → your email password or app password
RECIPIENT_ADDR → the email that will receive the alert
RECIPIENT_NAME → the name of the recipient (used in the message)
